覺得有趣好玩，於是我們大概嘗試實做看看

首先，我們先 Create 一個 backdoor_test.c 的檔案，內容如下 


#include <stdio.h>
#include <stdlib.h>

int main(void)
{
        puts("Yo");
        return 0;
}

$ gcc -c backdoor_test.c
