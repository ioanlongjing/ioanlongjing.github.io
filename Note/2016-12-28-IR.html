[ Note - IR ] 不專業幫人檢(看)測(看) Server 小小筆記

呃... 由於這禮拜要寫東西，可是我還是不知道要寫些什麼？(遮臉)
那... 就來筆記一下幫朋友看看他 Server 的事情吧

一開始是因為有點想偷懶，只有 Download Autoruns 跟 TCPView

= = = = = = = = = = = = = = = = = = = = = =

Autoruns :
開機檢測項目的工具，用來判斷裡面開機程序，如果不確定有沒有問題，就算確定有沒有問題，還有可以 sha1 後，扔到 VirusTotal 可以參考

TCPView :
檢視電腦的連線情況，列出本身使用中的 TCP 跟 UDP，而且當連線新增或關閉時，都會用顏色提醒

= = = = = = = = = = = = = = = = = = = = = =

好了，由於這篇不是在介紹 Tool，而且很容易就找到相關教學，關於軟體使用，就不再多說了

那... 一開始是打算藉由 Autoruns 檢查開機的項目，來檢測是否有問題，畢竟後門、木馬之類的必須確保一開機就會載入執行，覺得如果有中，從這邊看，應該至少能看到一些端倪
TCPView 是確認連線，如果沒有看到東西的話，想說可以從連線去判斷，例如把整台電腦斷網，在看有哪些 Process 還會嘗試連線，或其他不合理、不認識得連線，藉由這樣來縮小範圍， 有機會，還可以嘗試 try try 看、夢看看

記得當時看了 30 分鐘左右，還是沒有看到什麼特別有趣的東西，把一些看起來有問題的 Process，扔進 viristotal 也是正常
嗯.... 難道沒有問題？？？ 不太可能啊....（抓頭思考）

最後將那台 Server （或者是說一般的Windows 7）休眠重開，讓環境乾淨一點時（似乎因為他們跑了太多設計類得程式，讓整個電腦變得有點慢）
當開機到桌面時，忽然跳出一個視窗上面寫更新印表機還是什麼的, 接著就自動關掉, 連線的對象是我們親愛的隔壁國家.........
