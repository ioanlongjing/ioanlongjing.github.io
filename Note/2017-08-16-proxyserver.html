[ Note ] Ubuntu 架設 Proxy Server

不知道大家有沒有遇到一種狀況?
在沒有直接對外網路的環境中，忽然需要一台可以讓大家上網，又能管控流量跟帳號的 proxy server ?

於是乎，在稍微 Google，我覺得用 ubuntu 架 squid 看起來會比較方便
（ 關於 squid 的官網介紹請 按我 ）
所以在經過一番測試後，這邊先來筆記一下 Ubuntu 的環境中，要怎麼架設，跟在 F5 上，要怎麼掛起來

那... 一樣好了，由於本人實在有點懶，所以用 apt 直接做掉安裝的步驟 


 $   sudo apt-get install squid

好了，這樣我們就安裝好了
那今天的 Blog 就到此... (被揍)

在 squid 裡面，我們預設是只能自己連
如果要讓人家可以連的話，需要修改一下檔案，路徑為 


 /etc/squid3/squid.conf

那... 身為一個不專業的工程師，修改檔案前記得備份一份留存

那... 我們來 " 大家來找碴 " 一下
右邊的是我修改過後的版本，左邊得是它原來的版本
計時三秒鐘... (?)
