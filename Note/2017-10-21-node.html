[ Note ] 在 Ubuntu 上 install node 8.7.0

因為最近剛好在學 node.js, 但我們用超強的 apt-get install 後, 版本是 4.2.6 ...
現在都到 8.7.X 惹 QQ
於是找了很久, 終於讓找到方法了, 為了怕記憶力不好, 忘記力更強, 所以來筆記一下


首先我們先 install 跟 updata 一下 npm
可以用 curl 先下載跟用 bash 


curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.5/install.sh | bash

或著如果你沒有裝 curl 也可以用 wget 


wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.5/install.sh | bash

這會 clone 一份到 .nvm 中, 然後會新增到自己的 profile 中 (~/.bash_profile, ~/.zshrc, ~/.profile, or ~/.bashrc)
內容為.. 


export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This loads nvm

如果不確定我們有沒有成功在 nvm 上安裝可以嘗試看看 


$ command -v nvm
nvm

當成功安裝完那包得 nvm 後, 我們可以用這個 nvm 安裝 


nvm install node

接者我們在任何 shell 上, 都可以用到這個新版本
