[ Note ] 在 Ubuntu 上加指令

因為有點久沒自己加的指令到 Ubunut 上，目前剛好有機會就來筆記一下吧 XD

最近裝了個虛擬機，想在裡面裝個 sublime，但用 apt 安裝的東西是舊版的，所以想了想就手動安裝

那當然，我們先到 官網 下載



那我們開 terminal 手動解開一下


什..... 什麼？！


我們先執行看看...


好喔，所以這次不是包成 .deb 檔 （抹臉）
不過既然這樣，我們就手動自己加好惹

首先，我先把資料夾(sublime_3)改成隱藏資料夾(.sublime_text_3)


好惹，再來就是來做正事惹
我們先把 sublime 的執行檔加入 bash 可收尋的路徑
修改家目錄下的 .bashrc


第 120 行那個，把目前路徑加入到 PATH 裡面

再來把檔案本身 line 軟連結到 /usr/bin/ 裡面
PS : 如果用硬連結，就會只有那個檔案本身，會少了很多檔案，導致無法使用


看一下有沒有連結成功


歐，對了，在過程中，由於旁邊的 icon 會找不到
所以我們把那個 folder 裡面的 .desktop 檔，扔到 /usr/share/applications/ 裡面
