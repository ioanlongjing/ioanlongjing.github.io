[ Note ] mount 基本教學

好久沒寫 blog 了 XDDD
最近跟朋友討論了些關於 mount 的奇淫巧技，不過由於一些原因，目前還不太方邊弄出來
但又要寫 blog 了，所以先來寫一些關於 mount 的基礎教學吧 XD

首先我們先用 fdisk 這個指令看下目前的硬碟情況，所以我們先下 fdisk -l 這個指令


我們可以發現，剛剛上了一顆 200 G 還沒有來分割的硬碟（紅色圈起來的部份）

再來我們就準備分割硬碟了，一樣用 fdisk 這指令來分割硬碟，下的參數是後面接指定要格式話得硬碟
因為用 fidisk 這指令是可以互動式的指令，由於講解方便，就直接寫在圖片上了


之後我們用 sudo fdisk -l /dev/sdb 看一下剛剛的硬碟，應該會出現像這樣的訊息 


$ sudo fdisk -l /dev/sdb
Disk /dev/sdb: 200 GiB, 214748364800 bytes, 419430400 sectors
Units: sectors of 1 * 512 = 512 bytes
Sector size (logical/physical): 512 bytes / 512 bytes
I/O size (minimum/optimal): 512 bytes / 512 bytes
Disklabel type: dos
Disk identifier: 0xef03cdec

所用裝置   可開機 Start    結束    磁區  Size Id 類型
/dev/sdb1          2048 419430399 419428352  200G 83 Linux

我們這邊用 ext4 的格式把他 mount 起來到我們家目錄的一個叫做 DB 的資料夾 


sudo mount -t ext4 /dev/sdb1 /home/ioan/DB/

好惹，由於這顆目錄的 own 跟 group 都是屬於 root
我們這邊改一下 DB 這資料夾的 own 跟 group 


ioan@test:~$ sudo chown ioan DB

ioan@test:~$ sudo chgrp ioan DB
