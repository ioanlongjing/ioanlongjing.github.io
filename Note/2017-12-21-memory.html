[ Note ] 如何 dump 記憶體裡的資訊

今天假如我們看到 Windows 的特產藍天白雲
我們要如何從 memory 裡獲得我們要的資訊？

首先按下"開始"( start ) > "控制台"( Control Panel )


點選"系統安全性"( System and Security )


點選"系統"( System )


點選"進階系統設定"( Advanced system settings )


在進階的 Tab 裡，有個"啟動及修復"( Startup and Recovery )"的 Group 裡有個ｓ"設定"(Settings)


 點開後在"系統失敗"( System failure )的 Group 裡，只有無、小記憶體傾印(256KB)、核心記憶體傾印，沒有我們要的完整記憶體傾印


沒有的話怎麼辦麼？ 放棄 （x）
這時候我們就需要修改我們的註冊表

我們先再 Windows 的收尋欄位裡打上 "regedit"，收尋到一個檔案


會有 UAC 確認我們是不是真得是我們點選，直接按下確定即可
