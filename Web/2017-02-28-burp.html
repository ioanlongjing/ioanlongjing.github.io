[ Web ] Burp 教學 - 2 Brute Force
Web, brup

Hello ! 大家好 ~
還記得我們上次分享的 Burp 教學 - 1 嗎? 我們今天來點小小的實戰 - 暴力破解

首先我們先開啟 Burp ， 那... 由於詳細掛 proxy 跟上次一樣，這邊就不多贅訴上次的東西了


由於我們這邊只是示範，於是我架了 DVWA 來做示範，為了節省一些麻煩，我們先把安全性開到" low "


接下來，我們嘗試隨便打一點東西，讓 Burp 可以抓到 DVWA 的封包


稍微看一下，可以發現帳號密碼都先打 a 讓 Burp 抓到封包
再來我們把全部 POST 內容都先全選" Send to Intruder "


點開 Intruder ， 我們可以看到這邊新增了一個分頁 (2)


點開 positions 的選項，可以看到剛剛 POST 的詳細內容，還有很多被反白的內容
這邊反白的地方，意味著我們等等要跑字典檔去破解的參數


由於有些參數是連接的 token ，跟一些參數不需要動
所以我們先按 clear ，把全部 mark 的參數先 clear 掉，並且只把 username 跟 password 圈選起來，按 add


接下來選擇" payload " 選項，可以看到 payload set 有 1 跟 2 ，代表我們剛剛有選到兩個參數，等等可以用指定的方式跑 brute force


由於時間問題，我準備了一個小小的字典檔，不然會跑到海枯石爛


接著按下" load "把剛剛的內容 include 進來


Option 2 也是用一樣的方法 （當然，如果帳密都用不同的字典檔，當然也是可以的）
好了後按下右邊的" Start Attact "


接下來就能發現它開始跑各種組合 ＸＤ


那... 這邊我遇到了一個小問題，因為我有掛 WAF 在上面
所以當猜密碼的那邊猜到一定次數後被 WAF 擋下來
